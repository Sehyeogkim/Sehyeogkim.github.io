## CLI 기반 Agentic AI 도구들

Command Line Interface에서 사용할 수 있는 Agentic AI 도구들을 살펴보겠습니다.

### 주요 도구들

#### 1. Codex
- Anthropic의 Claude 기반 CLI 도구
- 코드 생성 및 수정에 특화

#### 2. Claude Code  
- Anthropic의 공식 CLI 도구
- 대화형 코드 작업 지원

#### 3. Gemini CLI
- Google의 Gemini 기반 도구
- 다양한 작업 자동화

### 사용 예시

\`\`\`bash
# Codex로 새로운 함수 생성
codex "create a function to parse JSON data"

# Claude Code로 코드 리뷰
claude code-review myfile.py
\`\`\`

## Key Takeaway
CLI 도구들은 개발자의 워크플로우를 크게 향상시킬 수 있습니다.
